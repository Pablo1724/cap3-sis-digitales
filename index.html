Álgebra de Boole

Fundada por George Boole, esta matemática es el lenguaje del hardware. Permite simplificar circuitos complejos mediante leyes lógicas de suma y producto.

3.1 Teoremas Fundamentales

Leyes de Identidad

A + 0 = A

A · 1 = A

Leyes de Nulidad

A + 1 = 1

A · 0 = 0

Leyes de De Morgan

(A + B)' = A' · B'

(A · B)' = A' + B'

Esencial para transformar compuertas NAND en NOR y viceversa.

3.4 Minterminos y Maxterminos

$\sum m$

Suma de Productos (Minterminos)

Se enfoca en las filas donde la salida es **1**. Cada variable se multiplica; si la entrada es 0 se niega, si es 1 se mantiene normal.

Ej: m1 (0,1) = A'B
$\prod M$

Producto de Sumas (Maxterminos)

Se enfoca en las filas donde la salida es **0**. Cada variable se suma; si la entrada es 1 se niega, si es 0 se mantiene normal.

Ej: M1 (0,1) = A+B'
